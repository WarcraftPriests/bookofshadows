## Cooldown Manager

Make sure you enable the Cooldown Manager itself in `Options` -> `Gameplay Enhancements` -> `Enable Cooldown Manager`. Then you can import my profiles by clicking `Advanced Cooldown Settings` -> `Import`.

I currently only support some specs/classes and I might add configurations for others in the future.

The below settings are just a starting point. I strongly encourage you to open up the Cooldown Manager (`/cdm`) and arrange spells in your desired order/place. The UI should automatically update as you move stuff around so feel free to experiment.

- Others: [SenseiClassResourceBar](https://www.curseforge.com/wow/addons/senseiclassresourcebar)
