Cooldown Manager​

Make sure you enable the Cooldown Manager itself in Options -> Gameplay Enhancements -> Enable Cooldown Manager. Then you can import my profiles by clicking Advanced Cooldown Settings -> Import.

I currently only support some specs/classes and I might add configurations for others in the future.
