Season 3 PTR Information

This guide is a temporary one-stop-shop for all current PTR info for testing Shadow Priest. This is all preliminary info, subject to change often. We encourage trying things out during testing as much as possible but feel free to use this as starting points.

This page will be updated throughout the PTR cycle and will be taken down once the patch goes live in favor of the normal guide sites.

Reminder to please post your thoughts and feedback on playing Shadow Priest on the forums. Be respectful.

Talents​

With the change to Shadow Crash it is now quite a stable talent for all builds, so we do not have as much build variety across raid encounters or even Mythic+. For now these seem to be the best starting points for each Hero Talent.

TUNING IS NOT FINAL: Most things are still very much a work-in-progress.

Voidweaver​

Core talents for Voidweaver are pretty static across fight types, with minor changes you can make. The multi-target build is particularly designed for the new 3 target council raid encounter.

  • Single Target
    • You can drop Phantasmal Pathogen for Voidtouched, this is about a 0.3% loss but can make it easier to play
    • If you want more execute/movement global between 35 and 20% you can play the Execute Build instead. This drops Voidwraith and Mindbender for Depth of Shadows and Deathspeaker. This is roughly a 0.2% loss in overall with a slight gain in execute.
  • Multi-Target
    • particularly designed for 3 target council raid encounter
  • Dungeons
    • identical to the multi-target build but we swap in Maddening Touch instead of Dark Thoughts. If you want to pickup Silence you would drop this point.

Archon​

Archon is just using a single build for the moment.

  • Triple Idol
    • We use this for everything. You can choose to swap out Mindbender for Deathspeaker if you want a bit more execute damage.
    • If you prefer to play with Voidtouched you can drop Phantasmal Pathogen for a 1.1% loss
    • For dungeons if you want to pickup Silence you would drop Phantasmal Pathogen

Rotation​

This season will be identical to the rotation of previous seasons, just with the loss of a few spells/procs that we did not always take.

Additions to the rotation:

  • Shadow Crash: used to mainly refresh DoTs. Can use above a normal Mind Flay filler action if you are capped on charges with Descending Darkness. Great global to use for movement similar to how we used Unfurling Darkness (assuming you have Descending Darkness).
  • Void Volley: Do not rush to press this, you have plenty of time and Void Torrent is cooling down as soon as you cast Void Torrent. Ideally this is a movement global when needed. If you do not have Void Blast or Mind Flay: Insanity to use you can send it.

Opener​

Voidweaver​

Pre-Cast Halo before combat to get 10 Insanity.

  1. Open with Shadow Crash if it will hit, otherwise Vampiric Touch
  2. Shadowfiend, Mindbender, or Voidwraith
  3. Mind Blast x1
  4. Dark Ascension
  5. Power Infusion + Tempered Potion + On Use Trinket
  6. Devouring Plague
  7. Void Torrent
  8. Void Blast x2

Archon​

  1. Open with Shadow Crash if it will hit, otherwise Vampiric Touch
  2. Shadowfiend or Mindbender
  3. Void Torrent
  4. Halo
  5. Void Eruption
  6. Power Infusion + Tempered Potion + On Use Trinket
  7. Void Bolt

Macros

You can use this macro to prevent double hitting Void Torrent cancelling itself with Void Volley.

#showtooltip
/stopmacro [channeling:Void Torrent]
/cast [known:Void Torrent] Void Torrent
